Mission Briefing

Project Overview

This project serves as a template and reference for students learning Object-Oriented Programming (OOP) and Software Quality. It simulates a battleship environment where players must strategically place ships and sink the enemy fleet.

A modern take on the classic naval warfare game, designed for the XVII century setting with updated software engineering patterns.

The game is played on a 10×10 grid. Hits are calculated based on the intersection of the shot vector and the ship's bounding box.

Installation & Setup

1

Clone the repository

git clone https://github.com/IGE-122994/Battleship2.git
2

Navigate to directory

cd Battleship2
3

Compile the project

mvn clean compile
4

Run unit tests

mvn test
5

Run with Docker

docker run --name battleship -it ige110323/battleship-game
